# Quickstart

Gemini Spark Task Board is a SaaS workspace for Gemini Spark task approval board.

## Fast Path

1. Open Gemini Spark Task Board and choose a bounded pilot workflow.
2. Enter public-safe examples for Gemini Spark task approval board.
3. Review the generated queue, evidence, or approval state with the owner.
4. Use pricing and checkout links when the pilot needs a paid workspace.
5. Export the resulting report or evidence for the team record.
